 Restaurant Management System – Backend
- Project Overview

The Restaurant Management System is a backend application designed to support restaurant owners in managing their operations efficiently. It provides secure APIs for handling authentication, menu management, order processing, payments, and email notifications.

This project demonstrates real-world backend development practices including authentication, payment gateway integration, and webhook handling.
